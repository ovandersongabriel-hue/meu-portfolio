Projeto B: Desafio Angular - Gerenciamento de Dados

1. Descrição e Desafios

Este projeto foi desenvolvido como parte de um desafio focado em Angular para simular uma aplicação de gerenciamento de dados. O objetivo era criar uma Single Page Application (SPA) robusta.

O desafio central foi a integração e o consumo de dados de uma API externa. A gestão do fluxo de dados assíncronos e reativos foi crucial, exigindo a implementação do padrão **Observable** da biblioteca **RxJS** para orquestrar as chamadas HTTP.

2. Soluções e Tecnologias

Utilizei o framework **Angular** para a estrutura da aplicação e **TypeScript** para garantir a tipagem e escalabilidade do código. Para o consumo da API, foram desenvolvidos *Services* dedicados que utilizam **HTTPClient** e manipulam as respostas via **RxJS Observables**, garantindo que os componentes da interface de usuário fossem atualizados de forma eficiente e reativa.

Tecnologias: Angular, TypeScript, RxJS, HTML, CSS.
